使用appledoc工具生成文档

1.安装brew

 (1)使用终端执行卸载命令：ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/uninstall)" 
   (注：若删除失败，可前往改文件夹手动删除：/usr/local)
 (2)使用终端执行安装命令：ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
 (3)升级brew：brew update 
             brew upgrade 
             brew cleanup 
 (4)因为OS X EI Capitan引入了系统完整性保护机制，升级后执行以下代码：sudo chown $(whoami):admin /usr/local && sudo chown -R $(whoami):admin /usr/local

3.安装appledoc工具：

  执行安装命令：brew install appledoc

4.安装完成后验证是否成功，执行命令：appledoc --version

5.生成文档(执行命令默认会编译出docset并安装进Xcode)

 (1)cd 到工程根目录
 (2)执行命令：appledoc --project-name MyProject --project-company ibireme ./

6.查看生成的文档：

  按住Opt键单击类名或方法名，就可显示出相应的帮助文档
